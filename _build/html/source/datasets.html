datasets.toy_dataset.generate_from_mixture(centers: list, num_per_class: list, n_dim=2)[source]¶

Generate gaussian mixture data

Parameters

  • centers – (list[list[float]) Coordinations of the centers of gaussian distributions. Each coordinate is represented by a list of float numbers. Each class cooresponds to a center.

  • num_per_class – (list[int]) Number of samples per class. Should have the same length with centers.

  • n_dim – (int) Dimension of the generated data, should be consistent with centers and num_per_class.

Returns:
data: (np.ndarray)

An 2 x n array representing n points generated by gaussain mixture.

targets: (np.ndarray)

An array of length n corresponding to the label of each point.
